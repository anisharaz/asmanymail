# AsManyMail

Create unlimited email addresses with persistent storage and full attachment support.

## Features

- **Unlimited Email Addresses** - Generate as many email addresses as you need instantly
- **Persistent Storage** - All emails stored permanently in the cloud
- **Full Attachments** - Send and receive files of all types
- **Instant Creation** - No waiting periods or verification required

## Tech Stack

- **Email Server**: Custom SMTP server with Docker
- **Frontend**: Next.js, React, TailwindCSS, shadcn/ui, Framer Motion
- **Backend**: Node.js, Prisma ORM
- **Database**: PostgreSQL
- **Auth**: Better Auth

## Project Structure

```
├── email-server/     # Custom email server with Prisma
└── web/              # Next.js frontend application
```

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm
- Docker (for email server)
- PostgreSQL database

## Usage

1. Visit the homepage and click "Get Started"
2. Create your account
3. Generate unlimited email addresses from your dashboard
4. Start receiving emails instantly
